1.はじめに
　ダウンロードしていただきありがとうございます。
　これは、Walls Luxorのアルファ版です。
　アルファ版の為、動作保証を負うことはできません。ご了承ください。

2.ソースコードについて
　まだまだ完成度は低いですが、こんなのでよろしければ是非改造にご利用ください。
　また、ソースコード上で何かアドバイスなどがありましたら
　コンタクトフォームなどで教えてくださるとありがたいです。
　
　※注意！改造する前に
　　[誰もが楽しめるものに]
　　　動作が重くなる、変なファイルが増えるなどの「ウイルス」と言ってもいいような機能は、
　　　利用者の方に迷惑がかかります。そのような機能を搭載し公開したい場合は、
　　　必ずその旨を伝えてから公開しましょう。
　　[読み込み時間]
　　　他人のことは言えませんが、やたら起動時間が長いとそれだけで飽きてしまうことがあります。
　　　最近の（本物の）OSは起動時間が短くなっています。
　　　「OSもどき」も、コレに倣って起動時間を短くしましょう。
　　[ソースはきれいに！]
　　　他人のことは言えませんが、あんまり汚く書くと今後作っていく上で必ず困ります。
　　　ソースコードを書くときは、似た処理やジャンルごとにまとめておくなどしておきましょう。

3.使用方法
　α版ですので、実行ファイルは同梱していません。
　Cities_Box.hspをコンパイルしてお使いください。
　
　コンパイルには「HSP スクリプトエディタ」が必要です。
　Walls_Luxor_Main.hspをHSPスクリプトエディタで開いてF5キーを押すか、
　メニューバーの「HSP(P)」→コンパイル(C)をクリックすると実行できます。
